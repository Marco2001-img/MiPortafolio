Expericia Laboral

  • Backend Api Reset Viajes

    Desarrollé una API REST para la gestión de reservas de viajes, permitiendo consultar viajes disponibles, apartar viajes y gestionar conductores. Utilicé .NET junto con Entity Framework y el patrón MVC para estructurar el proyecto y SQL Server como base de datos.

  • Sistema de Notas PubliG

    Implementé un sistema de notas y control financiero para la empresa PubliG utilizando Laravel en una arquitectura monolítica y una base de datos MySQL. Este sistema facilita la administración de clientes, control de notas y seguimiento financiero, y cuenta con un recordatorio automático de mensualidades,

  • Gestión de viajes de pasajeros

    la interfaz frontend creada en Angular de un sistema de gestión de viajes de pasajeros, enfocado en optimizar la administración y control de itinerarios. Implementé funcionalidades para el manejo eficiente de horarios, nombres de pasajeros y rutas, asegurando una experiencia de usuario fluida e intuitiva

  • Sistema de Tickets React

    Desarrollé un sistema de tickets para una empresa privada utilizando React y Redux Toolkit. Este sistema está diseñado con tres roles: cliente, administrador y agente, lo que permite gestionar solicitudes de soporte de manera eficiente. El sistema mejora significativamente el proceso de soporte técnico, optimizando la asignación de tickets y la comunicación entre usuarios y agentes, lo cual impacta directamente en la satisfacción del cliente.

  • Punto de Venta a 4 capas

    Desarrollé un sistema de punto de venta utilizando C# con una base de datos en SQL Server, generando un archivo instalable (.exe) para facilitar su distribución e instalación. El sistema incluye control de empleados con restricciones específicas según el rol del usuario, un módulo de proveedores, un módulo de administración de productos, y un rol de administrador para gestionar y supervisar las operaciones de la tienda. Este sistema fue diseñado para optimizar la gestión de la tienda, garantizando un control eficiente y organizado de sus procesos.
